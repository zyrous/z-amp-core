Class: CssMutator

.Components.AudioHtmlVisualiser. CssMutator

Mutates a specific CSS attribute of a single HTML element, based on a percentage value. If no mutation details are set, the default configuration will cause the HTML element's height to vary from 0 - 100%.

new CssMutator(htmlElement, cssProperty, unitOfMeasure, lowerValue, upperValue)

Construct a new mutator for a CSS property of an HTML element.

Parameters:
Name Type Default Description
htmlElement HTMLElement

The HTML element whose CSS property will be adjusted.
cssProperty String height

The property that will be mutated on the element. Defaults to "height".
unitOfMeasure String %

The unit of measure that will be appended to the CSS value. Defaults to "%".
lowerValue Number 0

The lower bounds of the value to set on the element. Defaults to 0.0.
upperValue Number 100

The upper bounds of the value to set on the element. Defaults to 100.0.
